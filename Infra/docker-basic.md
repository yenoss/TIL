## 2. IP address management(IPAM)

* 통하된 제품군 종단간 IP주소 인프라를 사용하도록 설정하는 도구. 자동으로 IPAM IP주소 인프라 서버와 네트워크 시스템 dns 서버를 검색.
* 다양한 장비간 IP 설정및 조회 하는것.
